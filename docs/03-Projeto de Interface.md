
# Projeto de Interface

Para o projeto de interface do Cor da Chita, foi optado por componentes visuais com poucos ornamentos e uma paleta de cores de baixo contraste, dando destaque ao conteúdo. Alguns elementos foram pensados, como um componente exibindo algumas citações relevantes ao branding da marca e ícones florais, para trazer ao projeto a identidade da marca.
